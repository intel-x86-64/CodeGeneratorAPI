# CodeGeneratorAPI
API for generating various matrix codes. Available for generation: UPC-A, UPC-E, EAN-8, EAN-13, Code 39, Code 93, Code 128, Codabar, ITF, QR Code, Aztec, DataMatrix, and PDF417.
